# Excel 模板格式参考

## 整体结构

导出 Excel 采用**转置格式**：每个参数占据一列，每行是一个元数据字段。

```
        |     A      |        B         |        C         | ...
Row 1   | (空)       | 参数名称1         | 参数名称2         | ...
Row 2   | 参数申请编码 | Y_FCC_XXX_001    | Y_FCC_XXX_002    | ...
Row 3   | 所属子领域  | 事项会计中台      | 事项会计中台       | ...
Row 4   | 所属应用编码 | DPMACCT          | DPMACCT           | ...
...     | ...        | ...              | ...              | ...
```

## 列说明

- **A列**: 元数据字段标签（固定43个字段）
- **B列及以后**: 每个参数一列，第1行为参数名称，后续行为各元数据字段值

## 样式规格

- **A列宽度**: 30.5 字符
- **B+列宽度**: 34.2+ 字符（根据内容自适应）
- **标题行**: 微软雅黑 11pt 加粗，浅蓝底 (#D9E2F3)，居中
- **标签列**: 微软雅黑 10pt，浅灰底 (#F2F2F2)
- **数据单元格**: 微软雅黑 10pt，白底
- **边框**: 细线，颜色 #B0B0B0
- **冻结窗格**: B2（固定首行和标签列）
- **自动换行**: 所有单元格启用
